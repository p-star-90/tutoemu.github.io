CEMU







1. Telecharger les resource

ICI

2. Extraire CEMU dans un dossier de votre choix

3. Extraire cemuhook dans un dossier de CEMU et ouvrez Cemu.exe

4. (optionnel) Vous pouvez choisir ou CEMU va stocker le coeur de la console

5. Vous devez choisir ou CEMU va chercher les jeux

6. Vous devez télécharger les pack graphique de la communauté

7. Vous devez cocher la case "chercher automatiquement les mise a jour"

8. (optionnel) Vous pouvez cocher la case "Demmarer les jeux en mode plain écran"

9. Vous devez cliquer sur "close"

10. Installer USB helper

Cliquer sur next chaque fois jusqu'a pouvoir choisire le version et choisisez "0.6.1.655"

11.(optionnel) cliquer sur "browse" pour choisir l'emplacement de USB Helper

12. cliquer sur "install" pour installer USB Helper puis ouvrez le

13. Accepter les CGU

14. Choisir une region

15. Accepter le "Disclaimer"

16. Cliquer sur "OK" et choisiser ou mettre vos jeux

17. Copier l'une des clé qui se trouve dans les resource et collez la dans l'espace prevu a cet effet puis cliquer sur "OK"

18. Allez dans filtre puis decocher toutes les case sauf celle sur l'image

19. Retournez dans "Bibliotheque" puis cherchez votre jeux et faite clique droit sur le jeux et cliquez sur "Télécharger le(s) jeu(x)"

20. (Optionnel mais très vivement conseillé) Cliquer sur "OK" pour telecharger la derniere mise a jour de jeux

21. (Optionnel) Cliquer sur "OK" pour télécharger les DLC (il y sont tous)

22. Cliquer sur "Télécharger le(s) jeu(x) en bas"

23. Très important : Cliquer sur "Décompress" (on le voit mieux quand USB Helper est en plein écran)

24. (optionnel) Dans CEMU, dans option, dans general, general : vous pouvez choisir une langue (vous devez redemarer CEMU pour voir votre laungue être appliqué) ici dans ce tutoriel, les oprion seront expliqué en français

25. Toujours dans les parametre, dans graphisme (dans l'image "Graphics") : choisir OpenGL sir vous possedez une carte graphique Nvidia sinon choisir Vulkan si vous avez une carte graphique AMD

26. Toujour au même endroit : choisir "Bilineair" (dans l'image "Bilinear") dans "Filtre d'agrandissment (upscale)" (dans l'image "Upscale filter")

27. Dans audio choisir dans moteur "XAudio2". Vous pouvez mettre une latence si le son du jeux bug

28. (Optionnel) vous pouvez modifier les paramettre "Overlay" et "Compte"

29. Dans la barre de navigation, CPU, Affinity mettre "All logical cores". Normalement c'est deja coché mais verifiez quand même

30. Dans la barre de navigation, Debug, Custom Timer mettre "QPC" et "1x speed". "1x speed "est deja coché mais verifiez quand même

31. Dans la barre de navigation, Debug, MM Timer Accuracy mettre "1 ms"

32. Dans la barre de navigation, Debug, mettre "Use Cemuhook H264"

33. Dans la barre de navigation, Option, Parametre des mannette : donner un nom au profile puis dans "Emulated controller" mettre "Wii U GamePad" puis cliquer sur le plus

34. Dans moteur, choisir "Keyboard" pour utiliser un clavier et choisir "XInput" pour choisir une manette

35. Dans Manette, choisir le peripherique en fonction de la selection dans moteur. Puis Cliquer sur "Ajouter" et la configuration des commande se fera automatiquement

36. Si ce n'est pa encore fait donner un nom au profile et cliquer sur sauvegarder

37. (Optionnel) vous pouvez cliquer sur reglage pour avoir acces a d'autre parametre notament le possibilité de mettre le vibreur sur la manette

38. (Optionnel mais vivement conseillé) Dans la barre de navigation, cliquer sur "Fichier" puis sur "Installer un titre de je, une mise a jour ou un DLC" puis allez chercher le dossier de la mise a jour téléchargé avec USB Helper, puis dans meta et choisir "meta.xml" et enfin patienter un petit moment

39. Pour installer un DLC; parreil mais cette fois il faut aller chercher "meta.xml" dans le dossier des DLC téléchargé avec USB Helper

39. Pour connecter une manette switch, premierement
